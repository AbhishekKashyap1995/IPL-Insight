1. Problem:
IPL fans and analysts struggle to compare players and understand performance trends across seasons using one clean, focused platform.

Why this works:

Specific (IPL only)
Clear user pain
Not trying to be ESPN or Cricbuzz


2. Target Users:

-IPL fans
-Fantasy cricket players
-Cricket analysts & bloggers


3. Core Features

-Player profiles with career & season stats
-Player vs Player comparison
-Match archive with results & venues
-Season-wise and team-wise performance
-Charts & trends (runs, averages, wickets)
-Fast search & filters

If a feature doesn’t fit here → it waits


4.  Out of Scope (for now)

-Live match scores
-Ball-by-ball commentary
-News & articles
-Non-IPL tournaments
-User chat / social features


5️. Define Success 

Success Criteria:

-Deployed and accessible online
-Can compare any two IPL players
-Accurate stats loaded from the database
-Mobile-friendly UI

6. Tech Stack

Frontend:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts

Backend:
- Node.js
- REST APIs
- Next.js Server Actions

Database:
- PostgreSQL
- Prisma ORM

Infrastructure:
- Vercel
- Supabase / Neon
- GitHub
